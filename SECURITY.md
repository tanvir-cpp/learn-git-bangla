## Reporting a Vulnerability

This is an educational documentation project and does not contain executable code. However, if you find any security concerns (such as malicious links, inappropriate content, or privacy issues), please report them by:

1. Opening a [GitHub Issue](https://github.com/tanvir-cpp/learn-git-bangla/issues)
2. Clearly describing the concern and its location in the project

We will review and address reported issues promptly.
